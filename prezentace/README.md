LaTeX template for presentations at FIT CTU
===================================

[![pipeline status](https://gitlab.fit.cvut.cz/theses-templates/FITpresentation-LaTeX/badges/main/pipeline.svg)](https://gitlab.fit.cvut.cz/theses-templates/FITpresentation-LaTeX/commits/main)


About
-----

Template for presentation for students and employees of [Faculty of Information Technology](https://fit.cvut.cz) at [Czech Technical University in Prague](https://cvut.cz). For use with XeLaTeX or LuaLaTeX.


Version
--------
1.0.1, built 7. 8. 2025. The whole changelog can be found in the [Changelog file](changelog.md).
